Приклад сторінки з розміткою TeX

В даному прикладі ми представимо таблицю похідних основних елементарних функцій. Для оформлення математичних формул ми будемо використовувати розмітку TeX.

Функція Похідна
$$f(x) = x^n$$ $$f'(x) = nx^{n-1}$$
$$f(x) = e^x$$ $$f'(x) = e^x$$
$$f(x) = \ln{x}$$ $$f'(x) = \frac{1}{x}$$
$$f(x) = \sin{x}$$ $$f'(x) = \cos{x}$$
$$f(x) = \cos{x}$$ $$f'(x) = -\sin{x}$$
$$f(x) = \tan{x}$$ $$f'(x) = \sec^2{x}$$
